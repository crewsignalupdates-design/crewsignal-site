Orientation

This hub links to a modular set of sourced reports explaining how the Communications Workers of America (CWA) is structured, how authority is distributed across sectors and districts, and how AFA operates within CWA’s constitutional framework.

Each report includes embedded primary sources so readers can independently verify the information presented.
