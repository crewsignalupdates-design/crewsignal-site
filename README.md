## Naming Conventions (Important)

To avoid broken links and deployment issues on case-sensitive systems
(Netlify / Linux), CrewSignal uses the following naming conventions:

### Directories
- lowercase only
- hyphen-separated
- no spaces

Example:
public/reports/contract-architecture/cwa-afa/air-transport-international/

### Files
- lowercase preferred
- hyphen or underscore separated
- no spaces

Examples:
index.html
frontier_cwa-afa_cba_2019-2024.pdf

### URLs
- must exactly match directory and file names
- avoid URL-encoded spaces (%20)
